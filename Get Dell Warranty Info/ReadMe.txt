This script utilizes Dell's TechDirect API to fetch warranty information based on device serial numbers. This script was exported to an .exe for ease of use by others at the company.

Device serial numbers are entered into a file which is then read by the script; the script then looks up device info based on that serial number and outputs the info to a file showing:
  - Serial Number
  - If device is covered under warranty
  - Level of service allowed under warranty
  - End date of warranty (if applicable)

See .docx file in same directory for instructions of using this script.
